# tele.chat PWA

Готовая устанавливаемая версия `tele.chat` для GitHub Pages.

## Публикация

Загрузите содержимое этой папки в корень репозитория `telechat`, затем включите GitHub Pages для ветки `main` и папки `/ (root)`.

После публикации приложение будет доступно по адресу:

`https://dark178098345-star.github.io/telechat/`

## Установка

- Windows/Android: открыть ссылку в Edge или Chrome и выбрать «Установить приложение».
- iPhone/iPad: открыть ссылку в Safari, нажать «Поделиться», затем «На экран Домой».
