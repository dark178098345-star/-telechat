# tele.chat v3

В этой версии добавлены:

- группы и каналы;
- приглашение и удаление участников;
- канал только для публикаций владельца;
- фотографии с автоматическим сжатием;
- голосовые сообщения до 45 секунд;
- новый компактный блок кнопок для ПК и телефона;
- красивый экран запуска.

## Как включить

1. Открой Supabase → SQL Editor → New query.
2. Открой файл `supabase-groups.sql`, скопируй его целиком и нажми Run.
3. Загрузи файлы этой папки в корень GitHub-репозитория `-telechat` с заменой старых.
4. Подожди зелёный статус GitHub Pages и перезапусти установленное приложение.

Если старая версия осталась в кэше, полностью закрой tele.chat и открой снова. При необходимости удали ярлык и установи приложение заново.
