# tele.chat v4

В этой версии добавлены:

- поиск людей, групп и каналов в одном поле;
- общие группы и каналы, которые можно найти по названию и вступить самостоятельно;
- приватные пространства, доступные только приглашённым участникам;
- переключение «Общая / Приватная» при создании и в информации о пространстве;
- приглашение и удаление участников;
- канал только для публикаций владельца;
- фотографии с автоматическим сжатием;
- голосовые сообщения до 45 секунд;
- адаптивный интерфейс для ПК и телефона.

## Обновление Supabase

Если таблицы групп уже создавались, выполни только файл supabase-privacy-v4.sql целиком в Supabase → SQL Editor → New query → Run.

Для чистой установки можно выполнить полный файл supabase-groups.sql.

После публикации полностью закрой установленное приложение и открой снова. Новый service worker удалит старый кэш.
