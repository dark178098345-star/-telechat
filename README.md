# tele.chat v5

В этой версии добавлены:

- описание «О себе» до 220 символов;
- красивый просмотр чужого профиля по нажатию на шапку личного чата;
- просмотр профилей участников группы;
- статус, аватар, имя, ник и время последнего посещения в карточке профиля;
- стабильный Realtime-индикатор «печатает…» в личных чатах и группах;
- поддержка нескольких одновременно печатающих участников;
- общий поиск и общие/приватные группы и каналы;
- фотографии и голосовые сообщения.

## Обновление Supabase

Выполни файл supabase-profile-v5.sql целиком в Supabase → SQL Editor → New query → Run.

После публикации полностью закрой установленное приложение и открой снова. Новый service worker удалит старый кэш.
